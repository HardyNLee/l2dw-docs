# 快捷键

## 全局

| 快捷键 | 说明 |
| --- | --- |
| F2 | 显示/隐藏背景 |
| F3 | 显示/隐藏 UI |
| F4 | 切换分辨率：`1280 x 720` `1920 x 1080` `2560 x 1440` |
| F5 | 重置镜头位置与缩放 |
| F6 | 为当前软件窗口截图，保存到 L2DW 软件目录下的 `Snapshots` 文件夹 |
| F7 | 为当前立绘截图（无滤镜），保存到 L2DW 软件目录下的 `Snapshots` 文件夹 |

## 主页

### 角色预览

| 快捷键 | 说明 |
| --- | --- |
| Ctrl F | 复制指令 - 表情动作 |
| Ctrl T | 复制指令 - 变换效果 |
| Ctrl A | 复制指令 - 表情动作 + 变换效果 |
| Ctrl X | 复制指令 - 表情动作 + 变换效果（分行） |

### 表情编辑

| 快捷键 | 说明 |
| --- | --- |
| Ctrl C | 导出表情到粘贴板 |

## 动作

| 快捷键 | 说明 |
| --- | --- |
| Ctrl C | 复制框选关键帧 |
| Ctrl V | 粘贴关键帧到播放头位置 |
| Delete | 删除框选关键帧 |
